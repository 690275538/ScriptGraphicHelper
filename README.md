# ScriptGraphicHelper

**一款简单好用的图色助手,  快速生成多种脚本开发工具的图色格式代码**

&nbsp;

**多平台分支**

&nbsp;

 **与AOT分支的不同**

无法进行AOT编译, 暂不支持句柄模式和图像编辑功能(后期可能会以dll的形式来支持win))

win: 支持

mac:  支持, debug阶段(aj连接模式, tcp模式)

linux: 未支持

&nbsp;

[下载地址](https://gitee.com/yiszza/ScriptGraphicHelper/releases)
